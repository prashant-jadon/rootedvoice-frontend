# Rooted Voices - Telehealth & Practice Management Platform

A beautiful, modern Next.js frontend demo showcasing a comprehensive telehealth and practice management platform for speech & language therapy.

## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Navigate to the dummy folder:
```bash
cd dummy
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
