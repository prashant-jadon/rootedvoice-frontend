## üåü Key Features by Category

### 1. AI-Powered Features ‚ú®

#### AI SOAP Note Generation
- **Location**: Video Call page
- **How it works**: After each session, AI automatically generates:
  - Subjective observations
  - Objective measurements
  - Assessment of progress
  - Plan for next session
  - Recommended activities
- **Benefits**: Saves 15-20 minutes of documentation time per session

#### AI Resource Search
- **Location**: Resource Library page
- **How it works**: Smart search understands natural language queries
  - "Find /r/ sounds for 5-year-old"
  - "Aphasia materials for adults"
  - "AAC assessments for non-verbal children"
- **Filters**: By age, goal, domain, diagnosis
- **Benefits**: Find materials 10x faster than manual search
